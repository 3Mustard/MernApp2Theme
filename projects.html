Kpop Merchandise Exchange

*Active Development

The purpose of this application is to allow international fans of Korean music to exchange merchandise and trading cards, create profiles, chat with other fans and organize purchases to save on shipping costs.

Although unfinished I am passionate about this project and want to share the resources I currently have.

The back end was built with JavaScript using the Node and Express libraries. Current features include CRUD options for User login, Authentication and Profiles.

I am currently studying how Socket.io works in order to build out chat functionality.

The front end will be built on the React.js Library to provide users with a pleseant and responsive experience.

Database services provided by MongoDB Atlas.

This project will be deployed with Heroku in the future. Stay tuned for more updates!
